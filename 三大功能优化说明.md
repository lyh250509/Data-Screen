## 🔄 使用方法

### 切换历史日期
1. 点击右侧面板的日期选择器
2. 只能选择3天内的日期（今天、昨天、前天）
3. 选择后，所有数据自动更新：
   - 顶部统计卡片
   - 3D场景中的人员/车辆标记
   - 左侧设备统计图表
   - 右侧人员流量趋势图
   - 右侧车辆类型统计

### 查看热力图
- 查看右侧热力图组件
- 观察渐变条和图例说明
- 实时统计每5秒自动更新

## 💡 技术亮点

### 1. 响应式历史数据
```javascript
// 监听日期变化，自动加载历史数据
watch(() => dataStore.selectedDate, (newDate) => {
  loadHistoricalData(newDate)
})
```

### 2. 数据衰减算法
```javascript
// 越早的数据量越少，模拟真实场景
personnelMultiplier = 1 - daysDiff * 0.15
vehicleMultiplier = 1 - daysDiff * 0.12
deviceMultiplier = 1 - daysDiff * 0.05
```

### 3. 动态热力统计
```javascript
// 每5秒更新峰值和平均密度
setInterval(() => {
  peakDensity = `${100 + random(50)}人/区`
  avgDensity = `${30 + random(20)}人/区`
}, 5000)
```
