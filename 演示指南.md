### 5. 告警系统
**说明**："系统实时监控各类异常，按级别分类显示"

**展示**：
- 高级告警：红色边框（火警、入侵等）
- 中级告警：橙色边框（温度异常等）
- 低级告警：蓝色边框（维护提醒等）

### 6. 技术架构
**说明**："项目采用Vue 3 + TypeScript + Vite构建"

**技术栈**：
- **前端框架**：Vue 3 Composition API
- **状态管理**：Pinia
- **图表库**：ECharts 5
- **地图**：OpenLayers 9
- **3D引擎**：Three.js
- **构建工具**：Vite
- **类型检查**：TypeScript

### 4. 常见问题准备

**Q: 这些数据是真实的吗？**
A: 目前使用的是模拟数据用于演示，实际项目中可以连接真实的后端API或WebSocket服务。

**Q: 可以定制化吗？**
A: 完全可以，包括数据类型、图表样式、颜色主题、布局结构等都可以根据需求定制。

**Q: 性能如何？**
A: 项目使用了Vue 3和Vite，具有出色的开发和运行性能。图表使用ECharts按需加载，3D场景使用Three.js优化渲染。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器（Chrome、Firefox、Safari、Edge等），推荐使用Chrome以获得最佳体验。

**Q: 可以移动端访问吗？**
A: 可以，但大屏主要针对桌面端和大屏显示器优化，移动端可能需要额外适配。
